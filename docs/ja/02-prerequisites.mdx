## Azure の要件

- Contributor アクセス権を持つ Azure サブスクリプション
- Azure CLI がインストールされ、認証済みであること（`az login`）
- Terraform >= 1.5.0

## SSH キー

VM へのアクセスには SSH キーペアが必要です：

```bash
ssh-keygen -t ed25519 -f ~/.ssh/id_ed25519
```
