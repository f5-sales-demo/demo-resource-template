## البدء السريع

<Steps>

1. استنساخ المستودع

   ```bash
   git clone https://github.com/f5-sales-demo/REPLACE_REPO_NAME.git
   cd REPLACE_REPO_NAME/terraform
   ```

2. نسخ ملف المتغيرات النموذجي

   ```bash
   cp terraform.tfvars.example terraform.tfvars
   ```

3. تعديل `terraform.tfvars` بالقيم الخاصة بك

   ```hcl
   subscription_id = "your-azure-subscription-id"
   # TODO: Add component-specific required variables
   ```

4. التهيئة والتخطيط

   ```bash
   terraform init
   terraform plan
   ```

5. التطبيق

   ```bash
   terraform apply
   ```

</Steps>
