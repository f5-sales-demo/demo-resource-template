## متطلبات Azure

- اشتراك Azure مع صلاحية المساهم (Contributor)
- تثبيت Azure CLI والمصادقة عليه (`az login`)
- Terraform >= 1.5.0

## مفتاح SSH

يلزم وجود زوج مفاتيح SSH للوصول إلى الجهاز الافتراضي:

```bash
ssh-keygen -t ed25519 -f ~/.ssh/id_ed25519
```
