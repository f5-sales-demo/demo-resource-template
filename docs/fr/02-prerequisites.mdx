## Exigences Azure

- Abonnement Azure avec accès Contributeur
- Azure CLI installé et authentifié (`az login`)
- Terraform >= 1.5.0

## Clé SSH

Une paire de clés SSH est requise pour l'accès à la VM :

```bash
ssh-keygen -t ed25519 -f ~/.ssh/id_ed25519
```
