## Démarrage rapide

<Steps>

1. Cloner le dépôt

   ```bash
   git clone https://github.com/f5-sales-demo/REPLACE_REPO_NAME.git
   cd REPLACE_REPO_NAME/terraform
   ```

2. Copier le fichier d'exemples de variables

   ```bash
   cp terraform.tfvars.example terraform.tfvars
   ```

3. Modifier `terraform.tfvars` avec vos valeurs

   ```hcl
   subscription_id = "your-azure-subscription-id"
   # TODO: Add component-specific required variables
   ```

4. Initialiser et planifier

   ```bash
   terraform init
   terraform plan
   ```

5. Appliquer

   ```bash
   terraform apply
   ```

</Steps>
