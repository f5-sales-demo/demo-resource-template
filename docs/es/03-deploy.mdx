## Inicio Rápido

<Steps>

1. Clonar el repositorio

   ```bash
   git clone https://github.com/f5-sales-demo/REPLACE_REPO_NAME.git
   cd REPLACE_REPO_NAME/terraform
   ```

2. Copiar el archivo de variables de ejemplo

   ```bash
   cp terraform.tfvars.example terraform.tfvars
   ```

3. Editar `terraform.tfvars` con sus valores

   ```hcl
   subscription_id = "your-azure-subscription-id"
   # TODO: Add component-specific required variables
   ```

4. Inicializar y planificar

   ```bash
   terraform init
   terraform plan
   ```

5. Aplicar

   ```bash
   terraform apply
   ```

</Steps>
