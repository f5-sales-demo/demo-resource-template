## Requisitos de Azure

- Suscripción de Azure con acceso de Contributor
- Azure CLI instalado y autenticado (`az login`)
- Terraform >= 1.5.0

## Clave SSH

Se requiere un par de claves SSH para el acceso a la VM:

```bash
ssh-keygen -t ed25519 -f ~/.ssh/id_ed25519
```
