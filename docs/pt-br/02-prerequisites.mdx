## Requisitos do Azure

- Assinatura do Azure com acesso de Colaborador
- Azure CLI instalado e autenticado (`az login`)
- Terraform >= 1.5.0

## Chave SSH

Um par de chaves SSH é necessário para acesso à VM:

```bash
ssh-keygen -t ed25519 -f ~/.ssh/id_ed25519
```
