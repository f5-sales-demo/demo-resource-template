## เริ่มต้นอย่างรวดเร็ว

<Steps>

1. โคลน repository

   ```bash
   git clone https://github.com/f5xc-salesdemos/REPLACE_REPO_NAME.git
   cd REPLACE_REPO_NAME/terraform
   ```

2. คัดลอกไฟล์ตัวอย่างตัวแปร

   ```bash
   cp terraform.tfvars.example terraform.tfvars
   ```

3. แก้ไข `terraform.tfvars` ด้วยค่าของคุณ

   ```hcl
   subscription_id = "your-azure-subscription-id"
   # TODO: Add component-specific required variables
   ```

4. เริ่มต้นและวางแผน

   ```bash
   terraform init
   terraform plan
   ```

5. นำไปใช้

   ```bash
   terraform apply
   ```

</Steps>
