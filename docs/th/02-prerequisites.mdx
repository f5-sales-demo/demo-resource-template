## ข้อกำหนดของ Azure

- Azure subscription ที่มีสิทธิ์ Contributor access
- ติดตั้งและยืนยันตัวตน Azure CLI แล้ว (`az login`)
- Terraform >= 1.5.0

## คีย์ SSH

จำเป็นต้องมีคู่คีย์ SSH สำหรับการเข้าถึง VM:

```bash
ssh-keygen -t ed25519 -f ~/.ssh/id_ed25519
```

## ประมาณการค่าใช้จ่าย

สิ่งที่ต้องทำ: อัปเดตด้วยค่าใช้จ่ายรายเดือนโดยประมาณของคอมโพเนนต์ของคุณ

| ทรัพยากร | ค่าใช้จ่ายรายเดือนโดยประมาณ |
|---|---|
| VM (REPLACE_SIZE) | REPLACE |
| Public IP | ~$4 |
| **รวม** | **REPLACE** |
