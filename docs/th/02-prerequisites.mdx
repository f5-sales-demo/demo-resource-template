## ข้อกำหนดของ Azure

- Azure subscription ที่มีสิทธิ์ Contributor access
- Azure CLI ติดตั้งแล้วและผ่านการยืนยันตัวตน (`az login`)
- Terraform >= 1.5.0

## SSH Key

จำเป็นต้องมีคู่คีย์ SSH สำหรับการเข้าถึง VM:

```bash
ssh-keygen -t ed25519 -f ~/.ssh/id_ed25519
```
