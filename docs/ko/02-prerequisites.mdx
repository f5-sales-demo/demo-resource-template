## Azure 요구 사항

- Contributor 액세스 권한이 있는 Azure 구독
- Azure CLI 설치 및 인증 완료 (`az login`)
- Terraform >= 1.5.0

## SSH 키

VM 접근을 위해 SSH 키 쌍이 필요합니다:

```bash
ssh-keygen -t ed25519 -f ~/.ssh/id_ed25519
```
