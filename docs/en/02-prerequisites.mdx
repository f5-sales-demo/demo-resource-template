## Azure Requirements

- Azure subscription with Contributor access
- Azure CLI installed and authenticated (`az login`)
- Terraform >= 1.5.0

## SSH Key

An SSH key pair is required for VM access:

```bash
ssh-keygen -t ed25519 -f ~/.ssh/id_ed25519
```
