## Requisiti Azure

- Sottoscrizione Azure con accesso Contributor
- Azure CLI installata e autenticata (`az login`)
- Terraform >= 1.5.0

## Chiave SSH

È necessaria una coppia di chiavi SSH per l'accesso alla VM:

```bash
ssh-keygen -t ed25519 -f ~/.ssh/id_ed25519
```
