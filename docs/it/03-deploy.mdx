## Avvio Rapido

<Steps>

1. Clonare il repository

   ```bash
   git clone https://github.com/f5xc-salesdemos/REPLACE_REPO_NAME.git
   cd REPLACE_REPO_NAME/terraform
   ```

2. Copiare il file di variabili di esempio

   ```bash
   cp terraform.tfvars.example terraform.tfvars
   ```

3. Modificare `terraform.tfvars` con i propri valori

   ```hcl
   subscription_id = "your-azure-subscription-id"
   # TODO: Add component-specific required variables
   ```

4. Inizializzare e pianificare

   ```bash
   terraform init
   terraform plan
   ```

5. Applicare

   ```bash
   terraform apply
   ```

</Steps>
