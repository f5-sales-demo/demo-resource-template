## Azure-Anforderungen

- Azure-Abonnement mit Contributor-Zugriff
- Azure CLI installiert und authentifiziert (`az login`)
- Terraform >= 1.5.0

## SSH-Schlüssel

Ein SSH-Schlüsselpaar ist für den VM-Zugriff erforderlich:

```bash
ssh-keygen -t ed25519 -f ~/.ssh/id_ed25519
```
