## Azure आवश्यकताएँ

- Contributor एक्सेस के साथ Azure सब्सक्रिप्शन
- Azure CLI इंस्टॉल और प्रमाणित (`az login`)
- Terraform >= 1.5.0

## SSH कुंजी

VM एक्सेस के लिए SSH कुंजी जोड़ी आवश्यक है:

```bash
ssh-keygen -t ed25519 -f ~/.ssh/id_ed25519
```
