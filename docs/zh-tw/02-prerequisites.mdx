## Azure 需求

- 具有 Contributor 存取權限的 Azure 訂閱
- 已安裝並完成驗證的 Azure CLI（`az login`）
- Terraform >= 1.5.0

## SSH 金鑰

存取虛擬機器需要 SSH 金鑰對：

```bash
ssh-keygen -t ed25519 -f ~/.ssh/id_ed25519
```
