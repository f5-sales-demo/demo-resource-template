## Azure 要求

- 具有参与者访问权限的 Azure 订阅
- 已安装并完成身份验证的 Azure CLI（`az login`）
- Terraform >= 1.5.0

## SSH 密钥

访问虚拟机需要 SSH 密钥对：

```bash
ssh-keygen -t ed25519 -f ~/.ssh/id_ed25519
```
